# SCHEMA DE BASE DE DATOS DE PRODUCTOS

## Información General
- Dataset: Productos de Electrodomésticos
- Total de registros: 300
- Descripción: Base de datos de productos electrodomésticos con información de precios, stock, características técnicas y disponibilidad.

id_producto, nombre, categoria, subcategoria, marca, precio_usd, stock, color, potencia_w, capacidad, voltaje, peso_kg, garantia_meses, descripcion
